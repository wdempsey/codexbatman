# Rubric: Tutor Mode

## Passing Standard

The output passes if it:

- identifies the underlying skill correctly
- preserves the base workflow
- converts delivery into scaffolded learning
- withholds full solutions by default
- asks for student reasoning at meaningful checkpoints

## Failure Modes

Fail if the output:

- simply executes the base skill normally
- gives the finished answer immediately
- asks vague pedagogical questions with no workflow structure
- removes important constraints from the wrapped skill
