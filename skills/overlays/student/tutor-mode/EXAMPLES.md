# Examples: Tutor Mode

## Example 1

User request:

> Help me do problem framing for my assignment, but don’t just do it for me.

Expected behavior:

- wrap `problem-framing`
- ask the student to define the objective type first
- provide a checkpoint instead of a finished framing artifact

## Example 2

User request:

> Use tutor mode on this project setup task. I want to learn how to think through the structure.

Expected behavior:

- wrap `project-setup`
- guide the student through identifying artifacts and gaps
- avoid creating the complete plan immediately
