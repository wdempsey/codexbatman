# Rubric: Hint Ladder

## Passing Standard

The output passes if it:

- identifies the blocked step
- starts with the least revealing useful hint
- escalates gradually
- prompts the student to attempt the next move

## Failure Modes

Fail if the output:

- skips directly to the solution
- gives hints unrelated to the actual workflow step
- keeps escalating when the problem is conceptual misunderstanding
