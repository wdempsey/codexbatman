# Rubric: Misconception Diagnosis

## Passing Standard

The output passes if it:

- ties the diagnosis to a concrete workflow step
- names a specific misconception
- explains why the misconception causes the observed error
- provides a corrective contrast and revision prompt

## Failure Modes

Fail if the output:

- just says the student is wrong
- jumps directly to the correct answer
- diagnoses a misconception when the issue is only lack of effort or missing information
