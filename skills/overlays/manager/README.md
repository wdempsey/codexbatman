# Manager Overlays

This directory contains overlays for the data science manager role.

- These overlays wrap canonical and manager workflow skills rather than replacing them.
- The purpose is to translate artifacts into executive summaries, project-state views, and communication workflows.
- Default behavior is to keep projects primary and communicate status, risk, and next steps clearly.

Available overlays:

- `executive-summary`
- `project-tracker`
- `communication-workflows`
