# Overlay Skills

This category contains cross-cutting overlay skills.

- Use these files to improve how Codex works across domains rather than to run a single domain workflow.
- Overlays handle prompt structuring, session capture, and continuous-improvement support files that other workflows can depend on.
- They are designed to compose with both data science and management skills without changing the underlying workflow definitions.
- `student/` adds tutoring overlays that wrap existing skills with scaffolded help, progressive hints, practice generation, and misconception diagnosis.
- `practitioner/` adds direct-execution overlays for working analysts who need artifact enforcement and minimal scaffolding.
- `manager/` adds summary, tracking, and communication overlays for project-centered management work.
