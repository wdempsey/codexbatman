# Rubric: Weekly Review

## Passing Standard

The output passes if it:

- states the review window and source coverage
- synthesizes multiple sources into a coherent project view
- distinguishes verified facts from uncertainty
- ties weekly activity to strategic objectives
- identifies owners, actions, blockers, and escalation needs

## Failure Modes

Fail if the output:

- reads like a raw chronology dump
- omits source gaps
- invents quantitative claims
- ignores the prior dashboard baseline
- buries urgent blockers or action items
