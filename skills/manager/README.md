# Manager Skills

This category contains management and coordination skills.

- Use these files for inbox triage, meeting preparation, scheduling, weekly review, to-do handling, project setup, and goal tracking.
- The goal is project-centered operational awareness across email, calendar, documents, and task systems.
- These skills support the data science manager role while remaining usable by individual contributors who need structured coordination workflows.
- Folder-based manager skills now live alongside command-style manager skills. The folder-based layer defines reusable Codex workflows for `project-setup`, `weekly-review`, `inbox-triage`, and `stakeholder-update`.
