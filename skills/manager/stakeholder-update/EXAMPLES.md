# Examples: Stakeholder Update

## Example 1

User request:

> Draft a short funder update from the latest weekly review. Focus on milestones, delays, and next steps.

Expected behavior:

- use the weekly review and dashboard as primary sources
- emphasize milestones and dependencies
- keep the tone external and concise

## Example 2

User request:

> Write a collaborator update for the team after this week’s meetings. Keep it operational.

Expected behavior:

- focus on decisions, owners, and immediate next steps
- avoid executive-summary language

## Example 3

User request:

> Prepare a leadership update with one paragraph on status, one on risks, and one ask.

Expected behavior:

- tailor the structure to leadership needs
- highlight the concrete ask clearly
- suppress lower-value operational detail
