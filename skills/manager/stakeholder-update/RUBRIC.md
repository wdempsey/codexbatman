# Rubric: Stakeholder Update

## Passing Standard

The output passes if it:

- is tailored to the stated audience
- uses current project artifacts as evidence
- communicates progress, risks, and next steps clearly
- includes an explicit ask when one is needed
- avoids unnecessary internal detail

## Failure Modes

Fail if the output:

- reads like an internal weekly log
- overstates certainty or progress
- includes unsupported facts
- ignores the audience’s decision needs
- omits material risks or dependencies
