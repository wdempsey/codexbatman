# Rubric: Data Audit

## Passing Standard

The output passes if it:

- documents the unit of analysis and source coverage
- performs quality, leakage, and bias checks
- addresses temporal validity when relevant
- states a clear proceed decision
- identifies concrete next actions

## Failure Modes

Fail if the output:

- skips source or schema context
- ignores leakage and target timing
- treats joins casually
- gives a vague recommendation instead of a proceed decision
