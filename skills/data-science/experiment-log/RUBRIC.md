# Rubric: Experiment Log

## Passing Standard

The output passes if it:

- records enough context to reproduce the run
- documents the intended change and rationale
- reports results and comparison clearly
- interprets results without overclaiming
- ends with a concrete decision

## Failure Modes

Fail if the output:

- omits seed, split, or dataset version
- logs only metrics with no context
- hides failed experiments
- lacks a decision outcome
