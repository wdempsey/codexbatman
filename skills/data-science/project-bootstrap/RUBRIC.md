# Rubric: Project Bootstrap

## Passing Standard

The output passes if it:

- identifies the existing workspace state accurately
- defines explicit locations for canonical artifacts
- preserves existing conventions where possible
- separates raw and derived work clearly
- leaves the project ready for `problem-framing`

## Failure Modes

Fail if the output:

- creates structure without inspection
- blurs raw and processed boundaries
- adds unnecessary scaffolding
- assumes the workspace is ready without artifact paths
