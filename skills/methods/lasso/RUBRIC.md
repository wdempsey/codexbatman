# Rubric: Lasso

## What Successful Understanding Looks Like

- The student can explain lasso as a sparse regularized regression method.
- The student can describe feature selection intuition without overstating it.
- The student can contrast lasso with ridge.
- The student can name instability with correlated predictors.

## Minimum Acceptable Understanding

- can say when lasso might be preferred to plain linear regression
- can explain what a zero coefficient means operationally
- can name one limitation of lasso

## Common Misconceptions

- lasso always finds the true important variables
- dropped variables never matter
- lasso is just ridge with different notation
- sparsity guarantees the best predictive model

## Readiness To Return To Workflow

- can explain when lasso is appropriate
- can name one tradeoff versus ridge or plain linear regression
- can describe one caveat, such as instability with correlated predictors
