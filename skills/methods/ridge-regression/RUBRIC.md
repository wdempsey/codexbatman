# Rubric: Ridge Regression

## What Successful Understanding Looks Like

- The student can explain ridge as linear regression plus shrinkage.
- The student can describe the bias-variance tradeoff in simple terms.
- The student can say why ridge helps with correlated predictors.
- The student knows ridge usually shrinks but does not select.

## Minimum Acceptable Understanding

- can name one situation where ridge is preferable to plain linear regression
- can explain what `lambda` does at a high level
- can say that ridge usually keeps all predictors in the model

## Common Misconceptions

- ridge performs feature selection
- smaller coefficients always mean worse predictions
- multicollinearity only matters for interpretation, never for prediction stability
- regularization is unnecessary once a linear model fits the training data well

## Readiness To Return To Workflow

- can explain when ridge is appropriate
- can name one tradeoff versus plain linear regression or lasso
- can describe one caveat, such as scaling needs or lack of feature selection
