# Rubric: Linear Regression

## What Successful Understanding Looks Like

- The student can explain that linear regression is a baseline method for continuous prediction.
- The student can describe what coefficient direction means.
- The student can explain residuals as model errors or misses.
- The student understands that linearity is an approximation.

## Minimum Acceptable Understanding

- can say when linear regression is appropriate
- can name one reason it is a baseline before more complex models
- can describe one limitation of the straight-line assumption

## Common Misconceptions

- coefficients prove causality
- a linear model must fit every point closely to be useful
- one bad residual means the method failed
- higher complexity is always better than a baseline

## Readiness To Return To Workflow

- can explain when linear regression is appropriate
- can name one tradeoff versus a more flexible method
- can describe one failure mode or caveat
