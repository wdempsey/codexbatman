# Rubric: Random Forest

## What Successful Understanding Looks Like

- The student can explain a random forest as many trees combined together.
- The student can describe nonlinearity and interactions at a high level.
- The student understands the interpretability tradeoff relative to linear models.
- The student knows random forest should be compared with a baseline rather than assumed to be the default.

## Minimum Acceptable Understanding

- can say when random forest is appropriate
- can explain one benefit over a linear baseline
- can name one cost, such as lower interpretability

## Common Misconceptions

- a random forest is just one large tree
- more flexible automatically means better workflow choice
- feature importance proves causality
- a baseline is unnecessary once a forest is available

## Readiness To Return To Workflow

- can explain when random forest is appropriate
- can name one tradeoff versus linear regression
- can describe one caveat, such as lower interpretability or importance-score misuse
