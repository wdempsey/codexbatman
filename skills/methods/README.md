# Methods Skills

Workflow skills define project steps. Method skills explain algorithms.

Use method skills when a workflow proposes a method and the learner is not yet familiar enough with that method to continue confidently.

These skills are teaching adapters for the student overlay. They pause the workflow long enough to explain the method, clarify its baseline role and tradeoffs, and then hand the learner back to the relevant workflow skill.

## Initial Methods

- `linear-regression`
- `ridge-regression`
- `lasso`
- `random-forest`
- `gradient-boosting`

## Notes

This library is intentionally incremental. The initial methods cover common supervised learning workflows, and additional methods will be added gradually over time.

New method skills should rely on trusted source materials such as ISLP, ISLR-Python, and Hands-On Machine Learning (`handson-ml3`) while staying concise, repo-native, and tutoring-oriented teaching adapters rather than pasted chapter summaries.
