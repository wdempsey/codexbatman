# Experiment Log

## Purpose

Tracking experiments is required for reproducibility, comparison, and decision quality.

Without a structured log, model iteration becomes difficult to audit and hard to interpret over time.

## Workflow

- record run
- record parameters
- record metrics
- record dataset version
- record conclusions

## Artifacts

- experiment log
- comparison table
- decision notes
