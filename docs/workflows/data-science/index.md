# Core Data Science Workflow

## Why Workflows Matter

AI makes analysis faster. Workflow discipline makes it trustworthy.

This system encodes best practices as reusable workflow phases so analytical work stays legible, reproducible, and reviewable across projects.

## The Canonical Workflow

1. project bootstrap
2. problem framing
3. data audit
4. exploratory analysis
5. modeling
6. evaluation
7. experiment logging

```text
project bootstrap
|
v
problem framing
|
v
data audit
|
v
EDA
|
v
modeling
|
v
evaluation
|
v
experiment logging
```
