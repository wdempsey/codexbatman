# Problem Framing

## Why It Matters

Many data science failures come from poorly framed problems.

If the decision target, metric, or estimand is unclear, downstream modeling can be technically correct but operationally wrong.

Problem framing creates the conditions for valid analysis, interpretable outputs, and usable decisions.

## Workflow

1. define decision
2. define metric
3. define estimand
4. define constraints
5. define evaluation criteria

## Artifacts

Artifacts produced:

- problem brief
- metric definition
- data requirements
- risk notes
