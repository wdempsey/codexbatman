# Data Audit

## Purpose

The data audit establishes whether a dataset is fit for analysis.

It is used to:

- understand dataset structure
- detect data quality issues
- detect leakage

## Workflow

- schema validation
- missingness analysis
- distribution checks
- join validation
- leakage detection

## Artifacts

Artifacts produced:

- data audit report
- schema summary
- quality flags
