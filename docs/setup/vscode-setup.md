# Set Up VS Code for Codex

VS Code is a free code editor that gives you a file explorer, a text editor, and a built-in terminal in one window.

You do not need to write code to use it. For many students, it is simply the easiest way to keep the repository, documents, and terminal visible at the same time.

## Step 1: Install VS Code

1. Go to [code.visualstudio.com](https://code.visualstudio.com/)
2. Download the installer for your operating system
3. Install and open VS Code

## Step 2: Open This Repository

1. Go to **File -> Open Folder**
2. Select this repository
3. Confirm the workspace is trusted

Codex works best when the full repository is open, not just one file.

## Step 3: Open the Terminal

Open the built-in terminal from **View -> Terminal**.

This is where Codex will run.

## Step 4: Launch Codex

In the terminal, run:

```bash
codex
```

You are now ready to work through the student path.
